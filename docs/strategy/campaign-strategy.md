# Save Les Bolstad — Campaign Strategy

## Executive Summary

Les Bolstad Golf Course at the University of Minnesota is scheduled for permanent closure. This campaign uses AI-orchestrated multi-agent coordination to run parallel workstreams across outreach, social media, community organizing, fundraising, legal research, and innovative membership models — all aimed at demonstrating the course is financially viable and community-essential.

## Phase 1: Research & Foundation (Weeks 1-2)

### Goals
- Understand exactly why the university is closing the course
- Map all stakeholders and decision-makers
- Research comparable "save our course" campaigns
- Assess legal and policy landscape
- Begin building supporter database

### Agent Assignments
| Agent | Phase 1 Task |
|-------|-------------|
| outreach | Research university decision-makers, alumni networks |
| legal-policy | U of M land use policies, closure process, reversal precedent |
| analytics | Benchmark comparable campaigns, establish baseline metrics |
| course-ops | Gather financial data on course operations, maintenance costs |
| content-creator | Draft campaign narrative and brand guidelines |

## Phase 2: Community Mobilization (Weeks 2-4)

### Goals
- Launch petition and social media presence
- Generate media coverage
- Build volunteer corps
- Begin outreach to key stakeholders

### Agent Assignments
| Agent | Phase 2 Task |
|-------|-------------|
| social-media | Launch Twitter/X, Reddit, Instagram campaigns |
| community | Launch petition, plan first event, recruit volunteers |
| outreach | Begin formal outreach to university administration |
| content-creator | Produce launch content: blog posts, press releases |
| analytics | Track early engagement, refine messaging |

## Phase 3: Demonstrate Viability (Weeks 4-8)

### Goals
- Present financial model showing course can be self-sustaining
- Launch $BOLSTAD token concept (or alternative membership model)
- Secure first corporate sponsors or grants
- Generate sustained media coverage

### Agent Assignments
| Agent | Phase 3 Task |
|-------|-------------|
| course-ops | Complete business plan with revenue projections |
| crypto-membership | Finalize token design, legal review, announce concept |
| fundraising | Launch crowdfunding, submit first grant applications |
| legal-policy | Complete nonprofit formation, token compliance review |
| outreach | Deliver formal proposal to university |

## Phase 4: Negotiation & Resolution (Weeks 8-16)

### Goals
- Negotiate operating agreement with university
- Launch membership/token system
- Secure operating funding
- Plan reopening

### Agent Assignments
| Agent | Phase 4 Task |
|-------|-------------|
| coordinator | Lead negotiation strategy |
| course-ops | Finalize operational plan with university input |
| crypto-membership | Launch token sale / membership system |
| fundraising | Close bigger sponsorships and grants |
| community | Plan reopening celebration |

## Key Risks

| Risk | Mitigation |
|------|-----------|
| University won't negotiate | Build overwhelming public pressure + financial case |
| Land already committed to development | Legal challenge + political pressure |
| Insufficient funding | Multiple revenue streams: tokens, sponsors, grants, memberships |
| Community apathy | Make it about more than golf: parks, recreation, green space |
| Token legal issues | Structure as utility token, get legal review early |

## Success Metrics

- **Petition signatures**: 10,000+ in first month
- **Social media**: 5,000+ followers across platforms
- **Media hits**: 10+ stories in local/national outlets
- **Fundraising**: $100K+ committed before university proposal
- **Community events**: 3+ events with 100+ attendees each
- **University meeting**: Formal sit-down within 8 weeks
